## Leituras importantes

- `AbstractService.save` e `AbstractService.findOne` vêm da camada base compartilhada; a resolução manual identifica a entidade concreta pelo service/repository/model usados no fluxo.
- Essas resoluções não substituem teste ou telemetria, mas deixam claro o efeito de banco esperado para revisão e operação.
